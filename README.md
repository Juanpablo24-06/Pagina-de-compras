# Página de Compras

Proyecto inicializado con React y Vite. Incluye un layout general con navegación y enrutamiento
básico para las secciones principales.

## Requisitos
- Node.js 18+

## Scripts
- `npm install` para instalar dependencias.
- `npm run dev` para levantar el entorno de desarrollo.
- `npm run build` para generar los artefactos de producción.
- `npm run preview` para previsualizar el build localmente.

## Rutas configuradas
- `/` página de inicio.
- `/fidelidad` punto de entrada para la nueva página de programa de fidelidad.
- `*` página de no encontrado.
